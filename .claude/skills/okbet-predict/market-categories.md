# 市场分类参考

快速识别市场类型及其对排名的潜在影响。

## 高影响力市场

### 政治类

| 关键词 | 示例 | 波动潜力 |
|--------|------|----------|
| election, president | Brazilian presidential election | 极高 |
| mayor, governor | Mayor of Bucharest | 高 |
| nominate, appoint | Trump nominate Fed chair | 高 |
| Maduro, Venezuela | Maduro out by... | 极高 |

### 地缘政治类

| 关键词 | 示例 | 波动潜力 |
|--------|------|----------|
| ceasefire, war | Russia x Ukraine ceasefire | 极高 |
| invade, military | U.S. invade Venezuela | 极高 |
| sanctions | - | 高 |

### 经济政策类

| 关键词 | 示例 | 波动潜力 |
|--------|------|----------|
| Fed, interest rate | Fed decreases 25 bps | 高 |
| inflation, CPI | - | 中高 |
| tariff | - | 高 |

### 加密货币类

| 关键词 | 示例 | 波动潜力 |
|--------|------|----------|
| Bitcoin, BTC + 价格 | Bitcoin reach $100,000 | 高 |
| ETH, Ethereum | - | 中高 |
| airdrop | lighter airdrop | 中 |

### 科技/商业类

| 关键词 | 示例 | 波动潜力 |
|--------|------|----------|
| market cap, largest | NVIDIA largest company | 中高 |
| IPO, acquisition | - | 中高 |
| Elon Musk, tweets | Musk post 420 tweets | 中 |

## 低影响力市场

### 体育类

| 类型 | 示例 | 说明 |
|------|------|------|
| NBA | Lakers vs Clippers | 结果随机，通常小仓位 |
| NHL | Bruins vs Canucks | 同上 |
| 足球单场 | Barcelona win | 同上 |
| NFL | Eagles vs Bills | 同上 |
| Spread/O-U | Cavaliers (-13.5) | 更高随机性 |

### 短期事件类

| 类型 | 示例 | 说明 |
|------|------|------|
| 单日价格 | BTC above $90k on Jan 3 | 随机性高 |
| 卡牌价格 | Kabuto card hit $100 | 小众市场 |

## 影响力评估矩阵

```
              高投资额        低投资额
            ┌───────────────┬───────────────┐
高影响力市场 │  决定性影响    │  潜在影响     │
            │  权重: ★★★★★  │  权重: ★★★    │
            ├───────────────┼───────────────┤
低影响力市场 │  中等影响     │  噪音         │
            │  权重: ★★      │  权重: ★      │
            └───────────────┴───────────────┘
```

## 快速判断规则

1. **含政治人名** → 高影响力
2. **含国家名 + 军事词汇** → 极高影响力
3. **含 Fed/利率** → 高影响力
4. **含 BTC/Bitcoin + 价格** → 高影响力
5. **含队名 vs 队名** → 低影响力（体育）
6. **含日期 + win** → 低影响力（单场体育）

## 对决市场优先级

分析对决时按以下优先级排序：

1. 双方投资额均 > $20 的政治/经济市场
2. 双方投资额均 > $10 的地缘/加密市场
3. 一方投资额 > $30 的任何高影响力市场
4. 其他对立仓位（作为补充提及）
